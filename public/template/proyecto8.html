Descripción

Este proyecto tiene como objetivo utilizar una Raspberry Pi 3 para controlar el encendido y apagado de un foco a través de un módulo relay 2CH 5VDC. La Raspberry Pi servirá como controlador principal, gestionando la alimentación del foco mediante el módulo relay, el cual actuará como un interruptor que se controla de forma digital a través de los pines GPIO de la Raspberry. Este proyecto es un ejemplo básico de automatización de tareas, que puede ser escalado para el control de múltiples dispositivos eléctricos.

Características Principales:

Raspberry Pi 3:
La Raspberry Pi 3 actúa como el cerebro del sistema, encargada de recibir órdenes y activar o desactivar el foco mediante los pines GPIO.

Características clave:
-Procesador ARM Cortex-A53 Quad Core.
-Pines GPIO configurables para enviar señales digitales.
-Sistema operativo Raspbian o cualquier distribución de Linux compatible.

Módulo Relay 2CH 5VDC:
El módulo relay de 2 canales actúa como un interruptor controlado eléctricamente, permitiendo encender y apagar dispositivos de mayor potencia (como un foco) con señales de bajo voltaje desde la Raspberry Pi.

Características clave:
-Tensión de entrada: 5VDC.
-Dos relays independientes para controlar dos dispositivos.
-Capaz de controlar cargas de corriente alterna y corriente continua.
-Aislado eléctricamente para proteger la Raspberry Pi.

Foco:
El foco representa la carga controlada. Al ser un dispositivo de mayor potencia, no se puede conectar directamente a la Raspberry Pi, por lo que se utiliza el módulo relay como intermediario.
Características clave:
Alimentación desde una fuente externa (220V o 110V dependiendo de la región).
Control de encendido y apagado mediante el relay.
