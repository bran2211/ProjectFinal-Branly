Descripción

Este proyecto consiste en desarrollar una aplicación web que genere tablas de multiplicar dinámicamente utilizando JavaScript, con la opción de exportar las tablas generadas en formato PDF. El usuario podrá personalizar el funcionamiento del generador mediante etiquetas o parámetros que controlan el rango de las tablas, si se listan o no, y los límites de las multiplicaciones. La herramienta será especialmente útil para estudiantes o cualquier persona que desee visualizar, imprimir o guardar tablas de multiplicar de forma sencilla y personalizada.

Características Principales:

Generador de Tablas con JavaScript:
El generador de tablas de multiplicar está programado en JavaScript. A través de una interfaz web simple, el usuario puede ingresar los parámetros necesarios para generar una tabla de multiplicar desde una base definida hasta un límite dado.

Características clave:
-Etiqueta --base: Define el número base para el cual se generará la tabla de multiplicar.
-Etiqueta --hasta: Especifica hasta qué número se deben calcular los múltiplos.
-Etiqueta --limite: Determina hasta qué número se multiplicará en cada tabla.
-Etiqueta --listar: Opción que indica si la tabla de multiplicar debe mostrarse en la pantalla antes de exportarse a PDF.

Exportación a PDF:
La aplicación incluye una funcionalidad que permite exportar las tablas generadas en un archivo PDF, lo que facilita su almacenamiento o impresión. Para esto, se utilizarán bibliotecas como jsPDF o html2pdf.

Características clave:
Exportación automática de la tabla generada en PDF.
Personalización del formato del PDF (estilo, colores, fuente).
Opción para guardar el PDF en una carpeta específica.

Etiquetas Personalizables:
El generador acepta varios parámetros para personalizar las tablas:
--base: Define el número para el que se creará la tabla de multiplicar (ejemplo: --base 5 genera la tabla del 5).
--hasta: Permite establecer el número final hasta donde se generarán las tablas (ejemplo: --hasta 10 genera las tablas hasta el 10).
--limite: Controla hasta qué número se realizarán las multiplicaciones en cada tabla (ejemplo: --limite 12 genera hasta el 12 en cada tabla).
--listar: Si está activado, muestra las tablas en pantalla antes de generar el PDF.

Interfaz Web Amigable:
Campos de entrada para las etiquetas --base, --hasta, --limite y un checkbox para --listar.
Botón para generar y exportar la tabla en PDF.
Diseño limpio y minimalista para facilitar la interacción del usuario.
