Descripción

Este proyecto consiste en una página web inicialmente construida en HTML y convertida a HBS (Handlebars) para mejorar la reutilización de componentes y la gestión de plantillas dinámicas. La página está alojada en la plataforma Railway, un servicio de despliegue en la nube que facilita el proceso de hosting y administración. La aplicación está configurada para correr en el puerto 8081, y utiliza un enfoque modular para optimizar la estructura del código, permitiendo una mejor organización y mantenimiento.

Características Principales:

Conversión de HTML a HBS:Conversión de HTML a HBS:
La página fue transformada de HTML estático a HBS, aprovechando la capacidad de Handlebars para usar plantillas dinámicas y parciales.

Despliegue en Railway: Despliegue en Railway:
La aplicación está alojada en Railway, facilitando la implementación y administración de la web en un entorno en la nube.

Estructura modular:
Gracias a HBS, se ha separado el header, footer y otros componentes reutilizables, lo que simplifica el desarrollo y mantenimiento del proyecto.

Uso del puerto 8081:
La aplicación está configurada para ejecutarse en el puerto 8081, lo que permite una personalización en el entorno de despliegue.

Diseño responsivo:
Mantiene un diseño responsivo optimizado para diferentes dispositivos, asegurando una experiencia de usuario óptima en móviles, tablets y computadoras.

Integración con Railway:
La página está totalmente integrada con Railway, lo que facilita actualizaciones y monitoreo continuo del estado de la aplicación.
