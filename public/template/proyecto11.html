Descripción

Este proyecto es una página web interactiva que facilita el cálculo de las principales fórmulas del Movimiento Rectilíneo Uniforme (MRU) y el Movimiento Rectilíneo Uniformemente Variado (MRUV). Está orientada a estudiantes o cualquier persona interesada en la física, permitiéndoles realizar cálculos relacionados con la velocidad, desplazamiento, tiempo, aceleración y otros parámetros clave de estos tipos de movimientos. El diseño es limpio y minimalista, y está estructurado con HTML y CSS, ofreciendo una experiencia de usuario simple y funcional.

Características Principales:

Formularios para cálculos:
La página incluye dos secciones principales, una para cálculos de MRU y otra para cálculos de MRUV, con formularios dedicados para cada tipo de movimiento.

Interfaz simple y ordenada:
Un diseño limpio basado en CSS, con un enfoque en la organización de los elementos para mejorar la experiencia del usuario.

Cálculos automáticos: Cálculos automáticos:
Los usuarios ingresan los valores correspondientes (como velocidad, tiempo o distancia) y la página devuelve los resultados del cálculo de manera instantánea.

Estilo visual atractivo:Estilo visual atractivo:
El uso de CSS asegura que la página tenga un aspecto moderno y atractivo, con un enfoque minimalista que facilita la navegación y la comprensión de la información.
